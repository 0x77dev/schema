# GraphQL Nexus Examples

All of these example can be run at once:

With Yarn:

```
git clone https://github.com/graphql-nexus/nexus.git
cd nexus
yarn
yarn examples
```

or NPM:

```
git clone https://github.com/graphql-nexus/nexus.git
cd nexus
npm i
npm run examples
```
